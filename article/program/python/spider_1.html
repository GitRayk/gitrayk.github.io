python爬虫入门(一)

2021-8-19

当我们访问一个网页的时候，就是先向该网站发送访问请求，之后获得响应，并由浏览器解析响应内容，最终呈现在用户面前
那么对于爬虫来说也是一样的道理，我们需要用代码来模拟发送请求的过程并获取响应（可由requests模块完成）， 之后解析响应的内容并寻找自己所需要的资源（可由bs4模块完成）

首先我们安装requests模块：pip install requests
其官方中文文档为：requests快速上手
向网站发送请求的方式分为两种，get和post, 前者比较常见，响应速度快；而后者则可以以表单形式上传参数，可以进行更复杂的处理
要使用这两种方法，最简单就是 requests.get(url) 或 requests.post(url)
它们会返回一个Response对象，直接打印该对象会显示一段包含响应码的文本，200表示为成功取得响应，4XX表示客户端错误，5XX表示服务器错误响应
Response有几种常用的方法或属性：

方法或属性意义
encoding表示响应的编码方式，可以直接修改其值
text表示响应的文本内容，如html网站的源码
content表示响应的二进制内容，对于二进制文件如图标来说应以这种成员取得数据
status_code表示响应码

事实上我们通常会在请求响应时使用更多的参数

  • headers 以dict的类型传入，用于设置访问的请求头
  • data 以dict的类型传入，可以用于post访问时提交表单数据，在控制台的XHR中即请求荷载字段
  • timeout 设置停止等待响应的时间，若不设置则可能会长时间的处于等待响应状态
  • cookies 请求时发送cookies到服务器，cookie也可以写在headers中
