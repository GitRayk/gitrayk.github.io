virtualenv - Python 虚拟环境

2023-4-16

virtualenv 介绍

在实际开发过程中，经常会遇到不同的项目依赖的 Python 版本不一致的问题
除了通过update-alternatives 命令来管理软件版本外，Python 还可以使用 virtualenv 来创建不同的虚拟环境

为了使用virtualenv，我们可以通过 apt 进行安装：

sudo apt install virtualenv

使用虚拟环境

首先我们需要创建一个虚拟环境，只需要使用如下命令：

virtualenv -p python解释器路径 虚拟环境名
之后便会在当前工作路径下新建一个与虚拟环境名同名的目录， 所有虚拟环境所需要的文件都包含在其中

要想使用指定的虚拟环境，可以使用点命令来使用虚拟配置：

source 虚拟环境路径/bin/activate
之后命令行的开头都会显示当前所使用的虚拟环境名

在虚拟环境中使用 pip 安装的包都不会被全局使用

退出及删除

当想要推出当前的虚拟环境时，只需要执行以下命令：

deactivate

如果一个虚拟环境不再被使用，只需要删除其虚拟环境目录即可
