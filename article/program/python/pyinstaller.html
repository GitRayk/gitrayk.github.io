将python文件打包成应用程序

2021-8-17

安装pyinstaller模块: pip install pyinstaller
在命令行中使用pyinstaller [opt] py文件即可完成打包，打包完成的文件可以在同一系统下使用，即使没有安装python
部分参数信息如下：

  • -F 仅生成一个可执行文件
  • -D 是与-F相对的，生成的是一整个完整的文件夹，也是pyinstaller的默认方式
  • -p 可以指定一个路径来让pyinstaller自己去找程序需要的资源
  • -i 可以指定一个文件作为生成的可执行文件的图标
  • -w 生成的文件不会显示命令行窗口（仅当GUI编程时使用，否则会因没有任何界面而报错）
  • -c 与-w相对，同时也是默认

在使用pyinstaller命令时会在当前工作目录生成一个dist目录、一个build目录和一个临时的spec文件， 最终生成的项目会被保存在dist目录中，其他的则删除不影响可执行文件的使用
spec文件会保存一些生成可执行文件时用到的参数信息，所以可以通过修改该文件内容的方式来生成， 修改后对该spec文件使用pyinstaller命令即可

使用 exit() 时程序出错
解决方法：使用 sys.exit()
