使用 matplotlib 进行数学计算以及绘图

安装： pip install matplotlib
matplotlib 官方文档

使用前： import matplotlib.pyplot as plt

  • plt.plot(*args, scalex=True, scaley=True, data=None, **kwargs)

    通常使用为 plt.plot(x, y)
    其中 x 和 y 都是一组list表示的数据。该方法在figure中绘制出所有数据表示的点
    在不指定更多参数的时候，它通常还会按顺序将点两两连成一线。
    也可以只写一组list，这样的话默认横坐标是从 0 开始递增 1

    指定样式: plt.plot(x, y, "ro")
    字符串中可以是不同字符的组合，如 "y" 表示黄色，"r" 表示红色，"b" 表示蓝色，"o" 表示仅作点，"+"表示仅作点并用+表示点，"--"表示虚线，等等
    还可以使用更完整的参数写法，下面两种写法是等价的:
    plot(x, y, 'go--')
    plot(x, y, color='green', marker='o', linestyle='dashed')

  • plt.hist(x, bins)

    hist 方法的参数实在太多了，但基本用法就是根据给出的 x 数据按数量作出直方图

    在使用 bins 参数后，若 bins 是一个数字，则表示直方图中的柱的最大数量, 即使它是空的！
    这意味着这更像制定了整个图的宽度，倘若一个柱的宽度为0.5，则在 bins = 4 的时候，整个直方图会分布在[1, 3]之间。
    此时如果只有3个直方图数据，那么会在中间空出0.5的宽度来

  • plt.scatter(x, y)

    基本功能是根据给出的坐标数据作出散点图

    参数 s/c/marker:
    指定点的大小/颜色/样式

  • plt.show()

    展示最近的一副plt图像, block参数可以指定是否在展示图像时阻塞代码，默认为True
    即使为False, 也会在程序运行结束时立即关闭窗口

  • plt.savefig(fname)

    将 figure 图像保存在文件中

  • plt.clf()

    清除当前 figure 的所有axes，但是不关闭这个 window，所以能继续复用于其他的 plot。
