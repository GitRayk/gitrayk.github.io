Pip使用

pip命令是在python中用于安装及维护python包的，当我们需要使用一个新的模块时，pip的使用是很有必要的

查看pip版本：pip --version
安装包：pip install package_name且默认为最新版本，可以通过pip install package_name==version来指定安装包的版本
升级包：pip install --upgrade package_name同样可以指定版本号
卸载包：pip uninstall package_name
列出已安装的包：pip list
查看可升级的包：pip list -o
升级pip：
linux: pip install --upgrade pip
windows: python -m pip install -U pip
查看更多pip指令：pip -h

在使用pip时还要面临一个问题，就是默认的下载源不在国内，所以下载请求经常超时或速度极慢，建议使用-i参数指定pip源并用--trusted-host参数信任该域名

国内常用的镜像有：

  1. 豆瓣：https://pypi.doubanio.com/simple/
  2. 清华：https://pypi.tuna.tsinghua.edu.cn/simple
故其具体代码为pip install -i https://pypi.doubanio.com/simple --trusted-host pypi.doubanio.com package_name
