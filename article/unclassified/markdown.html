一分钟学会Markdown

2021-10-3

Markdown 是一种轻量级标记语言，它允许人们使用易读易写的纯文本格式编写文档。
使用它编写的文档可以导出为 HTML 、Word、图像、PDF、Epub 等多种格式的文档。 而且Markdown文件本身也在一定程度上支持HTML的语法
其文档后缀为 .md, .markdown。学习和使用起来非常简单易懂。
算了，还是看菜鸟教程吧：菜鸟教程 - Markdown

标题

最常见的markdown标题使用的是 #

# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
需要注意的是符号 # 后面应该跟一个空格
其中四级标题和正文中的字号是一样大的

另一种写法是： 

我展示的是一级标题
=================

我展示的是二级标题
-----------------

段落

Markdown中段落内正文没有特殊的格式，直接编写文字就可以了。
要换行时，使用两个空格加一个回车或者两次换行
正文中的特殊字体： 

*斜体文本*
_斜体文本_
**粗体文本**
__粗体文本__
***粗斜体文本***
___粗斜体文本___

列表：

无序列表使用 * 、+、- 加一个空格来作开头
有序列表使用数字加小数点作为开头
列表中可以嵌套任意内容

引用和代码：

引用的部分内容使用 > 加一个空格来作开头
单个短的代码片段使用一组反引号 ` ` 将其包围
代码块使用1个制表符或4个空格作开头
代码块还可以使用3个连续的反引号``` code ```将其包围， 但此时第一行内容相当于对该段代码使用的语言的描述，并不会显示出来

链接和图片：

链接的写法：[链接名称](链接地址) 或者 <链接地址>
图片的写法：![alt 属性文本](图片地址)

表格：

Markdown使用 | 来分割单元格和作边框，使用若干个 - 来分割表头和其他数据
| 表头 | 表头 |
| ---- | ---- |
| 单元格 | 单元格 |
| 单元格 | 单元格 |
其中表头分割符的不同写法，会影响数据的排版：
:-- 数据会左对齐
--: 数据会右对齐
:-: 数据会居中对齐

其他技巧：

使用LateX改变字体颜色，语法为： $\color{颜色}{要变色的文字}$
颜色可以使用文字描述，也可使用 #FFFFFF 这种6位十六进制表示
